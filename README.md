# Messages API

## Maven
```XML
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>

<dependency>
    <groupId>com.github.SoyAldo</groupId>
    <artifactId>messages-api</artifactId>
    <version>{VERSION}</version>
</dependency>
```

## Tipos de acciones disponibles

### Sound
`[sound] sound_name yaw pitch`

### Title
`[title] fist line|second line`
